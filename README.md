# MySavings - 个人资产与储蓄管理助手

**MySavings** 是一款专为理财爱好者设计的 Android
资产管理工具。它不仅能帮助你记录每一笔银行存款及其预期收益，还能管理公积金、规划大额支出计划，让你对自己的“净资产”了如指掌。

## 🌟 核心功能

### 1. 资产全景看板

* **多维度汇总**：实时计算“本息合计”、“公积金合计”以及“总净资产”。
* **隐私保护**：支持一键隐藏敏感金额。
* **资产下钻**：支持按年、月查看详细的存款分布和收益明细。

### 2. 智能储蓄管理

* **利息预估**：输入本金、利率和存期，自动计算到期本息。
* **到期提醒**：清晰展示每笔存款的起始日期与预计到期时间。
* **便捷操作**：支持侧滑删除记录，点击 FAB 快速新增。

### 3. 支出计划提醒

* **自定义事件**：记录父母生日、节假日支出等大额计划。
* **倒计时预警**：在首页显著位置提醒即将到来的支出事项。
* **周期循环**：支持每年重复提醒（如生日、纪念日）。

### 4. 公积金追踪

* **独立账户**：单独管理公积金余额与每月汇缴额。
* **资产融合**：将公积金纳入总资产统计，反映真实的个人财务状况。

---

## 🛠️ 技术栈

* **语言**：Kotlin
* **架构**：MVVM (Model-View-ViewModel)
* **UI 组件**：Material Design 3, ViewBinding, ConstraintLayout
* **异步处理**：Kotlin Coroutines & Flow (实现数据的实时感知刷新)
* **数据存储**：Room Database (本地 SQLite 存储)
* **导航**：Jetpack Navigation Component

## 🚀 快速开始

1. **克隆仓库**：

```bash
git clone https://github.com/endlessYoung/MySavings.git

```

2. **导入 Android Studio**：
   选择 `File > Open` 并选择项目根目录。
3. **运行项目**：
   点击 `Run 'app'` 并在模拟器或实体机上查看。
